# 🚀 Cogni Advisor

Cogni Advisor is a modern AI-powered web application built with **Next.js (App Router)**, **TypeScript**, and **Yarn**.  
The project focuses on performance, scalability, and clean architecture to deliver intelligent and seamless user experiences.

---

## 🧠 About The Project

Cogni Advisor is designed to provide a smart and interactive platform experience.  
Built using modern frontend best practices, the application ensures high performance, maintainability, and production readiness.

---

## 🛠 Tech Stack

- Next.js 14 (App Router)
- React
- TypeScript
- Yarn
- Tailwind CSS
- ESLint
